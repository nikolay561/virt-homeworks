`my.cnf` в директории /etc/mysql.Измените его согласно ТЗ (движок InnoDB):- скорость IO важнее сохранности данных;- нужна компрессия таблиц для экономии места на диске;- размер буффера с незакомиченными транзакциями 1 Мб;- буффер кеширования 30% от ОЗУ;- размер файла логов операций 100 Мб.Приведите в ответе изменённый файл `my.cnf`.## Ответ:```[mysqld]skip-host-cacheskip-name-resolvedatadir=/var/lib/mysqlsocket=/var/run/mysqld/mysqld.socksecure-file-priv=/var/lib/mysql-filesuser=mysql# Set default enginedefault-storage-engine = InnoDB# IO speed upinnodb_flush_method = O_DSYNCinnodb_flush_log_at_trx_commit=2# Allow table compressioninnodb_file_per_table = 1# Set transaction bufferinnodb_log_buffer_size = 1M# Cache buffer 30% RAM sizeinnodb_buffer_pool_size = 1177M# Log bufferinnodb_log_file_size = 100Mpid-file=/var/run/mysqld/mysqld.pid[client]socket=/var/run/mysqld/mysqld.sock!includedir /etc/mysql/conf.d/```---### Как оформить ДЗВыполненное домашнее задание пришлите ссылкой на
